---
title: "Telefonanruf tätigen"
api: "POST /user/make_call"
description: "Einen Telefonanruf tätigen"
---

Dieser Endpunkt ermöglicht es Ihnen, einen Telefonanruf zu tätigen.

### Request-Body

<ParamField body="phone_number" type="string" required>
  Die Telefonnummer des Kunden im E.164-Format (z. B. +1234567890)
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  Die ID des Assistenten, mit dem der Anruf getätigt werden soll
</ParamField>

<ParamField body="variables" type="object">
  Die Variablen, die an den Lead übergeben werden sollen
  
  <Expandable title="variables-Eigenschaften">
    <ParamField body="customer_name" type="string" default="<string>">
      Der Name des Kunden
    </ParamField>
    
    <ParamField body="email" type="string" default="<string>">
      Die E-Mail des Kunden
    </ParamField>
  </Expandable>
</ParamField>

### Antwort

<ResponseField name="message" type="string" default="Call initiated successfully">
  Die Nachricht der Antwort
</ResponseField>

<ResponseExample>

```json 200 Antwort
{
  "message": "Call initiated successfully",
  "data": {}
}
```

</ResponseExample>
