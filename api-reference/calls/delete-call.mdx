---
title: "Anruf löschen"
api: "DELETE /user/calls/{call}"
description: "Einen bestimmten Anrufdatensatz löschen"
---

Dieser Endpunkt ermöglicht es Ihnen, einen bestimmten Anrufdatensatz zu löschen, der dem authentifizierten Benutzer gehört.

### Pfad-Parameter

<ParamField path="call" type="integer" required>
  Die eindeutige Kennung des zu löschenden Anrufs
</ParamField>

### Antwort

<ResponseField name="message" type="string">
  Bestätigungsnachricht, die anzeigt, dass der Anruf erfolgreich gelöscht wurde
</ResponseField>

### Fehlerantworten

<ResponseField name="404 Not Found">
  <Expandable title="Fehlerantwort">
    <ResponseField name="message" type="string">
      Fehlermeldung, wenn der Anruf nicht gefunden wurde oder dem authentifizierten Benutzer nicht gehört
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Erfolgsantwort
{
  "message": "Call deleted successfully"
}
```

```json 404 Not Found
{
  "message": "Call not found"
}
```

</ResponseExample>

### Hinweise

- Nur Anrufe, die zu Assistenten gehören, die dem authentifizierten Benutzer gehören, können gelöscht werden
- Sobald ein Anruf gelöscht wurde, kann er nicht wiederhergestellt werden
- Diese Aktion entfernt den Anrufdatensatz dauerhaft, einschließlich aller zugehörigen Transkripte, Aufzeichnungen und Metadaten
- Die Anrufaufzeichnungsdatei (falls vorhanden) wird ebenfalls aus dem Speicher gelöscht
