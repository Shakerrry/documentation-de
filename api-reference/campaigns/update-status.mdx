---
title: "Kampagnenstatus aktualisieren"
api: "POST /user/campaigns/update-status"
description: "Eine Kampagne im Autocalls-System starten oder stoppen"
---

Dieser Endpunkt ermöglicht es Ihnen, eine Kampagne im Autocalls-System zu starten oder zu stoppen.

### Anfragekörper

<ParamField body="campaign_id" type="integer" required>
  Die ID der zu aktualisierenden Kampagne
</ParamField>

<ParamField body="action" type="string" required>
  Die Aktion, die für die Kampagne ausgeführt werden soll. Muss entweder "start" oder "stop" sein
</ParamField>

### Antwort

<ResponseField name="message" type="string" default="Campaign started successfully.">
  Die Nachricht der Antwort
</ResponseField>

<ResponseField name="success" type="boolean" default="true">
  Ob die Operation erfolgreich war
</ResponseField>

<ResponseField name="data" type="object">
  Die Daten der Antwort
  
  <Expandable title="Daten-Eigenschaften">
    <ResponseField name="campaign_id" type="integer">
      Die ID der aktualisierten Kampagne
    </ResponseField>
    
    <ResponseField name="status" type="string" default="in-progress">
      Der neue Status der Kampagne (z. B. "in-progress" für gestartete Kampagnen, "stopped" für gestoppte Kampagnen)
    </ResponseField>
  </Expandable>
</ResponseField>
