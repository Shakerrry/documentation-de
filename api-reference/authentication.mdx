Die Autocalls-API verwendet API-Schlüssel zur Authentifizierung von Anfragen. Sie müssen Ihren API-Schlüssel im Header jeder Anfrage an die API einfügen.

## Einen API-Schlüssel erhalten

So erhalten Sie einen API-Schlüssel:

1. Melden Sie sich bei Ihrem Autocalls-Konto unter [https://app.autocalls.ai](https://app.autocalls.ai) an
2. Navigieren Sie zur Seite "API Keys"
3. Klicken Sie auf die Schaltfläche "Create new API Key"
4. Kopieren Sie Ihren neuen API-Schlüssel und speichern Sie ihn sicher

<Warning>
  Halten Sie Ihren API-Schlüssel vertraulich und sicher. Teilen Sie ihn nicht öffentlich und übertragen Sie ihn nicht in Versionskontrollsysteme.
</Warning>

## Verwendung des API-Schlüssels

Fügen Sie Ihren API-Schlüssel in den `Authorization`-Header jeder Anfrage ein:

```
Authorization: Bearer YOUR_API_KEY
```

Ersetzen Sie `YOUR_API_KEY` durch Ihren tatsächlichen API-Schlüssel.

## Beispielanfrage

Hier ist ein Beispiel, wie Sie den API-Schlüssel in eine cURL-Anfrage einbinden:

```bash
curl -X GET "https://api.autocalls.ai/api/user/campaigns" \
     -H "Authorization: Bearer YOUR_API_KEY"
```

Ersetzen Sie `YOUR_API_KEY` durch Ihren tatsächlichen API-Schlüssel bei Anfragen.
