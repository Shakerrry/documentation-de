---
title: "Eingehenden Assistenten-Webhook aktivieren"
api: "POST /user/assistants/enable-inbound-webhook"
description: "Eingehende Webhook-Benachrichtigungen für einen bestimmten Assistenten aktivieren"
---

Dieser Endpunkt aktiviert eingehende Webhook-Benachrichtigungen für einen Assistenten und ermöglicht es Ihnen, Echtzeit-Updates über eingehende Anrufabschlüsse und Daten zu erhalten.

### Anfragekörper

<ParamField body="assistant_id" type="integer" required>
  Die ID des Assistenten, für den der Webhook aktiviert werden soll
</ParamField>

<ParamField body="webhook_url" type="string" required>
  Die URL, an die Webhook-Benachrichtigungen gesendet werden
</ParamField>

### Antwort

<ResponseField name="message" type="string">
  Erfolgsmeldung, die bestätigt, dass der Webhook aktiviert wurde
</ResponseField>

<ResponseField name="data" type="array">
  Leeres Array (für zukünftige Verwendung reserviert)
</ResponseField>

### Fehlerantworten

<ResponseField name="404 Not Found">
  <Expandable title="Fehlerantwort">
    <ResponseField name="message" type="string">
      Fehlermeldung, wenn der Assistent nicht gefunden wurde oder dem authentifizierten Benutzer nicht gehört
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="422 Validation Error">
  <Expandable title="Fehlerantwort">
    <ResponseField name="message" type="string">
      Fehlermeldung, die einen Validierungsfehler anzeigt
    </ResponseField>
    <ResponseField name="errors" type="object">
      Detaillierte Validierungsfehler für jedes Feld
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Erfolgsantwort
{
  "message": "Webhook enabled successfully",
  "data": []
}
```

```json 404 Not Found
{
  "message": "Assistant not found"
}
```

```json 422 Validierungsfehler
{
  "message": "The given data was invalid.",
  "errors": {
    "assistant_id": [
      "The assistant id field is required."
    ],
    "webhook_url": [
      "The webhook url field is required."
    ]
  }
}
```

</ResponseExample>
