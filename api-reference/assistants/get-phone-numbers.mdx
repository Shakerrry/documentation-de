---
title: "Verfügbare Telefonnummern abrufen"
api: "GET /user/assistants/phone-numbers"
description: "Alle verfügbaren Telefonnummern für Assistentenzuweisung abrufen"
---

Dieser Endpunkt gibt eine Liste von Telefonnummern zurück, die dem authentifizierten Benutzer gehören und Assistenten zugewiesen werden können, mit optionaler Filterung nach Assistententyp.

### Abfrageparameter

<ParamField query="type" type="string" optional>
  Telefonnummern nach Assistententyp filtern. Optionen: `inbound`, `outbound`
</ParamField>

### Antwortfelder

<ResponseField name="data" type="array">
  <Expandable title="Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung der Telefonnummer
    </ResponseField>

    <ResponseField name="phone_number" type="string">
      Die Telefonnummer im E.164-Format
    </ResponseField>

    <ResponseField name="country_code" type="string">
      Der Ländercode für die Telefonnummer
    </ResponseField>

    <ResponseField name="type_label" type="string">
      Die menschenlesbare Typbezeichnung für die Telefonnummer
    </ResponseField>

    <ResponseField name="is_available" type="boolean">
      Ob die Telefonnummer für die Zuweisung verfügbar ist (derzeit keinem eingehenden Assistenten zugewiesen)
    </ResponseField>
  </Expandable>
</ResponseField>

### Hinweise

- Es werden nur Telefonnummern zurückgegeben, die dem authentifizierten Benutzer gehören
- Wenn `type=inbound` angegeben ist, werden Anrufer-ID-Nummern und bereits zugewiesene Nummern herausgefiltert
- Das Feld `is_available` gibt an, ob die Nummer einem neuen eingehenden Assistenten zugewiesen werden kann
- Telefonnummern werden im Values-Collection-Format zurückgegeben (indiziertes Array)
- Verwenden Sie das Feld `id` beim Zuweisen von Telefonnummern zu Assistenten
