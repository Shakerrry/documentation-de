---
title: "Debugging Runs"
icon: "bug"
description: "Stellen Sie sicher, dass Ihre Geschäftsautomatisierungen ordnungsgemäß ausgeführt werden"
---

Sie können jeden Run überwachen, der aus einem aktivierten Flow resultiert:

1. Gehen Sie zum Dashboard, klicken Sie auf **Runs**.
2. Finden Sie den Run, nach dem Sie suchen, und klicken Sie darauf.
3. Sie sehen den Builder im Nur-Ansicht-Modus, jeder Schritt zeigt ein ✅ oder ein ❌, um seinen Ausführungsstatus anzuzeigen.
4. Klicken Sie auf einen dieser Schritte, Sie sehen die **Eingabe** und **Ausgabe** im **Run Details**-Panel.
