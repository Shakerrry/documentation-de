## Trigger

Der Startpunkt des Flows bestimmt seine Ausführungshäufigkeit. Es gibt verschiedene Arten von Triggern, wie Schedule Trigger, Webhook Trigger oder Event Trigger basierend auf einem spezifischen Service.

## Aktion

Aktionen folgen nach dem Flow und steuern, was passiert, wenn der Flow aktiviert wird, wie das Ausführen von Code oder die Kommunikation mit anderen Services.
