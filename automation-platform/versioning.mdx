---
title: "Versionsverlauf"
icon: "clock"
description: "Erfahren Sie, wie Flow-Versionierung in Website Excellence Builders.ai funktioniert"
---

Website Excellence Builders.ai verfolgt alle veröffentlichten Flows und ihre Versionen. So funktioniert es:

1. Sie können einen Flow beliebig oft im **Entwurfs**-Modus bearbeiten.
2. Sobald Sie mit Ihren Änderungen fertig sind, können Sie ihn veröffentlichen.
3. Der veröffentlichte Flow wird **unveränderlich** sein und kann nicht bearbeitet werden.
4. Wenn Sie versuchen, einen veröffentlichten Flow zu bearbeiten, erstellt Website Excellence Builders.ai einen neuen **Entwurf**, falls keiner vorhanden ist, und kopiert die **veröffentlichte** Version in die neue Version.

Das bedeutet, Sie können immer zu einer vorherigen Version zurückkehren und den Flow im Entwurfsmodus bearbeiten, ohne die veröffentlichte Version zu beeinflussen.
