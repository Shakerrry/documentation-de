Im Folgenden finden Sie einige beliebte Tools, die Sie mit Autocalls.ai über die integrierte No-Code-Plattform oder direkte Integrationen verbinden können:

## 1. Google Sheets

- **Leads importieren**: Ein Sheet mit Telefonnummern für eine ausgehende Kampagne abfragen.
- **Anrufergebnisse protokollieren**: Transkripte oder Anrufzusammenfassungen an ein "Anrufhistorie"-Sheet anhängen.

## 2. Kalender (Cal.com, Google Calendar, etc.)

- **Terminplanung**: Die KI kann freie Zeitfenster vorschlagen und dann Ereignisse erstellen.
- **Erinnerungen**: Wenn ein Anruf über einen bevorstehenden Termin geht, kann die KI bestätigen oder neu planen.

## 3. HubSpot / CRMs

- **Lead-Daten**: Post-Call-Informationen können in einem Kontaktdatensatz gespeichert werden.
- **Status-Updates**: Lead-Status automatisch auf "Interessiert" oder "Benötigt Follow-up" ändern.

## 4. Slack / E-Mail-Benachrichtigungen

- **Sofortige Alarme**: Wenn ein Benutzer sagt "Ich möchte stornieren" oder "Ich möchte kaufen", einen Slack-Ping auslösen.
- **Tägliche Zusammenfassung**: Ihrem Team täglich eine E-Mail-Zusammenfassung von Anrufen, Ergebnissen und neuen Leads senden.

## 5. Andere (Facebook-Leads, etc.)

- Einige fortgeschrittene Benutzer integrieren mit Facebook Lead Ads oder anderen Marketing-Plattformen, um Echtzeit-Leads in eine Autocalls.ai-Outbound-Kampagne einzuspeisen.

---

**Tipp:** Im Autocalls.ai-Automatisierungseditor sehen Sie vorgefertigte Blöcke für viele beliebte Tools. Wenn etwas fehlt, können Sie oft **Webhooks** oder die **API** verwenden, um eine Verbindung herzustellen.
