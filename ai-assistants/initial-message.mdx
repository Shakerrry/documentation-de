Die ersten Sekunden eines Anrufs sind entscheidend - sie bestimmen, ob der Kunde in der Leitung bleibt. Sie haben zwei Optionen für die erste Begrüßung Ihres Assistenten: textbasierte Anfangsnachricht oder benutzerdefiniertes Anfangsaudio.

## Anfangsnachricht

Dies ist das Erste, was Ihr KI-Assistent sagt, wenn er einen Anruf beginnt. Die Nachricht wird genau so vorgelesen, wie sie geschrieben ist, daher:

### Best Practices

1. **Halten Sie es kurz**

   - Ziel sind 5-10 Sekunden
   - Kommen Sie schnell zum Punkt
   - Vermeiden Sie lange Unternehmensvorstellungen

2. **Schreiben Sie genau wie benötigt**

   - Fügen Sie korrekte diakritische Zeichen ein (é, ñ, ü usw.)
   - Verwenden Sie Interpunktion für richtige Pausen
   - Schreiben Sie Zahlen so, wie sie gesprochen werden sollen

3. **Beispielformate**

   ```
   Gut:
   "Hallo! Hier ist Sarah von ABC Company. Wie kann ich Ihnen heute helfen?"

   Besser mit Pausen:
   "Hallo! Hier ist Sarah von ABC Company... Wie kann ich Ihnen heute helfen?"

   Mit diakritischen Zeichen:
   "¡Hola! Soy María de ABC Company. ¿Cómo puedo ayudarte?"
   ```

## Anfangsaudio

Für den besten ersten Eindruck können Sie eine vorab aufgenommene Audiodatei verwenden:

### Vorteile

- Professionelle Qualität
- Perfekte Aussprache
- Menschliche Wärme
- Konsistente Wiedergabe
- Höhere Kundenbindung

### Einrichtungsprozess

1. Nehmen Sie Ihre Begrüßung mit einem professionellen Sprecher auf
2. Laden Sie die Audiodatei in den [Assistenteneinstellungen](/ai-assistants/settings/general#call-flow-configuration) hoch
3. Klonen Sie dieselbe Stimme für den Rest des Gesprächs
4. Aktivieren Sie die Anfangsaudio-Wiedergabe

### Best Practices

1. **Aufnahmequalität**

   - Verwenden Sie professionelle Ausrüstung
   - Nehmen Sie in einer ruhigen Umgebung auf
   - Halten Sie eine gleichmäßige Lautstärke ein
   - Speichern Sie in hochwertiger Formatierung

2. **Stimmanpassung**

   - Verwenden Sie denselben Sprecher zum Klonen
   - Bewahren Sie konsistenten Ton und Stil
   - Passen Sie Energieniveaus an

3. **Inhaltsrichtlinien**
   - Halten Sie es unter 10 Sekunden
   - Fügen Sie Firmennamen hinzu
   - Geben Sie den Zweck klar an
   - Klingen Sie einladend

### Beispiel-Skriptstruktur

```
[Begrüßung] + [Firmenname] + [Zweck/Frage]

"Hallo! Hier ist ABC Company, wir rufen bezüglich Ihrer kürzlichen Anfrage an. Wie geht es Ihnen heute?"
```

## Beide Methoden kombinieren

Sie können beides einrichten:

- Anfangsaudio als primäre Begrüßung
- Anfangsnachricht als Backup
- System verwendet Audio, wenn verfügbar

## Testen

Bevor Sie live gehen:

1. Rufen Sie Ihren Assistenten an
2. Achten Sie auf:
   - Klare Aussprache
   - Natürliche Pausen
   - Richtige Lautstärke
   - Reibungsloser Übergang zum KI-Gespräch

## Sprachüberlegungen

- Jede Sprache benötigt ihre eigene Anfangsnachricht/Audio
- Verwenden Sie Muttersprachler für Aufnahmen
- Berücksichtigen Sie regionale Akzente
- Testen Sie mit Zielgruppe

---

**Profi-Tipp:** Nehmen Sie mehrere Versionen Ihres Anfangsaudios auf und testen Sie, welche bessere Antwortquoten erzielt.
