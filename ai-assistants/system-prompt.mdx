Der System-Prompt ist die wichtigste Komponente Ihres KI-Assistenten. Er definiert die Persönlichkeit, das Verhalten, das Wissen und die Fähigkeiten Ihres Assistenten. Betrachten Sie ihn als das "Gehirn" und "Trainingshandbuch" kombiniert.

## Schnellstart mit Vorlagen

Um schnell zu beginnen:

1. Gehen Sie zu Ihren [Assistenteneinstellungen](/ai-assistants/settings/prompt-and-tools#system-prompt)
2. Finden Sie das System-Prompt-Feld
3. Klicken Sie auf die Schaltfläche "Vorlagen" in der Nähe
4. Wählen Sie eine Vorlage, die zu Ihrem Anwendungsfall passt
5. Passen Sie sie an Ihre Bedürfnisse an

## Sprachunterstützung

Ihr System-Prompt kann in jeder Sprache geschrieben werden, unabhängig von der gesprochenen Spracheinstellung:

- Schreiben Sie den Prompt in Ihrer bevorzugten Sprache
- Stellen Sie die gesprochene Sprache separat in den [Assistenteneinstellungen](/ai-assistants/settings/general#language-configuration) ein
- Die KI folgt den Anweisungen des Prompts, während sie in der ausgewählten Sprache spricht

Zum Beispiel:

- System-Prompt auf Englisch, gesprochene Sprache auf Spanisch eingestellt
- System-Prompt auf Deutsch, gesprochene Sprache auf Französisch eingestellt
- System-Prompt auf Chinesisch, gesprochene Sprache auf Englisch eingestellt

## Warum sie wichtig sind

Ihr System-Prompt:

- Formt, wie die KI denkt und antwortet
- Definiert Gesprächsgrenzen
- Bietet wesentliches Wissen
- Steuert Verhalten und Ton
- Bestimmt den Umgang mit Grenzfällen

## Hauptkomponenten

Ein guter System-Prompt sollte beinhalten:

### 1. Rolle & Identität

```
Sie sind ein professioneller Vertriebsmitarbeiter für [Unternehmen]. Sie sind spezialisiert auf [Produkt/Dienstleistung] und verfügen über umfassendes Wissen über unser Angebot.
```

### 2. Gesprächsstil

```
Bewahren Sie einen freundlichen, professionellen Ton. Verwenden Sie klare, prägnante Sprache. Vermeiden Sie Fachjargon, es sei denn, er wird ausdrücklich verlangt.
```

### 3. Wichtige Informationen

```
Unsere Hauptprodukte sind:
- Produkt A ($X/Monat): [Merkmale]
- Produkt B ($Y/Monat): [Merkmale]
Aktuelle Aktion: 20% Rabatt auf die ersten 3 Monate
```

### 4. Verhaltensrichtlinien

```
- Überprüfen Sie immer die Kundeninformationen, bevor Sie Kontodetails besprechen
- Leiten Sie an einen menschlichen Agenten weiter, wenn der Kunde frustriert erscheint
- Machen Sie keine Versprechen über Liefertermine
```

### 5. Antwort-Framework

```
Wenn nach Preisen gefragt wird:
1. Verstehen Sie zuerst ihre Bedürfnisse
2. Präsentieren Sie relevante Paketoptionen
3. Erklären Sie das Wertversprechen
4. Teilen Sie alle anwendbaren Rabatte mit
```

## Best Practices

1. **Seien Sie spezifisch**

   - Klare Anweisungen führen zu besseren Ergebnissen
   - Fügen Sie Beispiele guter Antworten hinzu
   - Definieren Sie, was NICHT zu tun ist

2. **Struktur ist wichtig**

   - Organisieren Sie Informationen logisch
   - Verwenden Sie Aufzählungszeichen und Abschnitte
   - Halten Sie zusammengehörige Informationen zusammen

3. **Testen und Iterieren**
   - Beginnen Sie mit einem einfachen Prompt
   - Testen Sie verschiedene Szenarien
   - Verfeinern Sie basierend auf Anrufaufzeichnungen
   - Fügen Sie Behandlung für Grenzfälle hinzu

## Häufige Fehler

- **Zu vage**: "Seien Sie hilfreich und professionell" (Nicht spezifisch genug)
- **Zu starr**: Jede mögliche Antwort skripten (Reduziert natürlichen Fluss)
- **Informationsüberflutung**: Unnötige Details einbeziehen
- **Fehlende Richtlinien**: Nicht angeben, wie mit häufigen Situationen umgegangen werden soll

## Beispielstruktur

```
# Rolle und Zweck
[Definieren Sie, wer die KI ist und ihre Hauptziele]

# Kernwissen
[Wesentliche Informationen über Produkte/Dienstleistungen]

# Gesprächsrichtlinien
[Wie mit Kunden zu interagieren ist]

# Antwortmuster
[Wie mit spezifischen Situationen umzugehen ist]

# Einschränkungen und Grenzen
[Was die KI NICHT tun sollte]
```

## Ihren Prompt testen

1. Führen Sie Testanrufe durch, die Folgendes abdecken:

   - Häufige Szenarien
   - Grenzfälle
   - Schwierige Situationen
   - Verschiedene Kundenpersönlichkeiten

2. Überprüfen und anpassen:
   - Hören Sie sich Anrufaufzeichnungen an
   - Überprüfen Sie die Antwortgenauigkeit
   - Überprüfen Sie die Tonkonsistenz
   - Testen Sie die Wissensspeicherung

## Optimierungstipps

- Beginnen Sie mit Fast Engine für schnelle Iterationen
- Verwenden Sie Anrufaufzeichnungen, um Lücken zu identifizieren
- Fügen Sie Beispiele guter/schlechter Antworten hinzu
- Fügen Sie Behandlung für unerwartete Fragen hinzu

---

**Profi-Tipp:** Ihr System-Prompt ist ein lebendiges Dokument - aktualisieren Sie ihn regelmäßig basierend auf tatsächlichen Anruferfahrungen und Kundeninteraktionen.
