---

Wenn Sie eine ausgehende Kampagne starten, können Sie wählen, welche Nummer auf den Telefonen der Empfänger angezeigt wird.

## 1. Dedizierte Nummer

- Wenn Sie eine Orts- oder gebührenfreie Nummer gemietet haben, können Sie diese als "Telefonnummer" in den [Assistenten-Einstellungen](/ai-assistants/settings/general#phone-number-configuration) auswählen.
- Gut für Markenkonsistenz oder wenn Sie auch eingehende Anrufe auf derselben Leitung annehmen.
- **Keine OTP-Verifizierung erforderlich** für Nummern, die über Website Excellence Builders.ai gemietet wurden.

## 2. Verifizierte bestehende Nummer (nur Anrufer-ID)

- Wenn Sie möchten, dass Ihre persönliche oder Firmentelefonleitung angezeigt wird, verifizieren Sie diese mit einem **OTP** oder kurzen Anruf.
- Dies behandelt keine eingehenden Anrufe für die KI, aber der Lead sieht diese erkannte Nummer auf der Anrufer-ID.
- **Hinweis**: OTP-Verifizierung ist NUR erforderlich, wenn Sie Ihre eigene bestehende Nummer als Anrufer-ID verwenden.

## 3. SIP-Integration

- Wenn Sie eine PBX- oder VOIP-Plattform verwenden, können Sie eine benutzerdefinierte Anrufer-ID von Ihrem bestehenden System übergeben.
- Stellen Sie sicher, dass Ihr Trunk oder Anbieter so eingestellt ist, dass eine bestimmte ausgehende Anrufer-ID zulässig ist.

## 4. Länderspezifische Regeln

- Einige Regionen verhindern die Verwendung von Nummern, die Sie nicht besitzen, als Anrufer-ID.
- Stellen Sie immer sicher, dass Sie das Recht haben, diese Nummer anzuzeigen, um Spam oder rechtliche Probleme zu vermeiden.

---

**Tipp:** Testen Sie, indem Sie Ihr eigenes Telefon oder einen freundlichen Kollegen anrufen, um zu bestätigen, dass die korrekte Anrufer-ID angezeigt wird, bevor Sie eine große Kampagne starten.
