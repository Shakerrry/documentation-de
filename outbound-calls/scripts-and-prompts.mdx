Die Kraft ausgehender Anrufe liegt in einem klaren, gut strukturierten Gesprächsablauf. So erstellen Sie ein effektives Skript.

## 1. Best Practices für System-Prompts

- **Stellen Sie Ihr Unternehmen vor**: _"Sie sind der Vertriebs-Bot für Ace Marketing…"_
- **Zweck**: _"…Sie bieten einen Rabatt von 20%, wenn der Benutzer interessiert ist, oder vereinbaren einen Rückruf, wenn er unsicher ist."_
- **Einwandbehandlung**: _"…wenn sie sagen 'kein Budget', versuchen Sie, einen Grund zu erfassen oder eine Erinnerung für nächsten Monat zu setzen."_

## 2. Beispiel (Verkaufs-Prompt)

```plaintext
"Sie sind der Vertriebs-Bot für ACME Corp. Begrüßen Sie höflich, erwähnen Sie die neue Produkteinführung und bieten Sie einen Rabatt von 10%, wenn der Lead interessiert ist. Wenn sie nach mehr Informationen fragen, laden Sie sie ein, einen Anruf mit einem menschlichen Agenten zu vereinbaren. Wenn sie ablehnen oder Desinteresse zeigen, beenden Sie den Anruf höflich."
```

## 3. Beispiel (Inkasso-Prompt)

```plaintext
"Sie sind der Inkasso-Bot für ABC Finance. Begrüßen Sie den Benutzer, verifizieren Sie seinen Namen und das letzte Zahlungsdatum. Wenn sie bereit sind zu zahlen, erfassen Sie Zahlungsdetails oder vereinbaren Sie einen Rückruf. Wenn sie ablehnen, notieren Sie höflich den Grund und beenden Sie den Anruf."
```

## 4. Umgang mit Ja/Nein/Vielleicht

- **Ja**: Details angeben oder an einen Menschen zur Finalisierung weiterleiten.
- **Nein**: Respektvoll abschließen, Lead als nicht interessiert markieren.
- **Vielleicht**: Follow-up-Anruf anbieten oder mehr Informationen bereitstellen.

---

**Tipp:** Halten Sie Ihr Skript kurz und direkt. Die KI kann sich anpassen, aber ein prägnanter Prompt hilft, Anrufe fokussiert und effizient zu halten.
