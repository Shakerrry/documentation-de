Schnelle Lösungen für häufige Lead-Management-Probleme und Importprobleme.

## Lead-Importprobleme

### Import schlägt komplett fehl
**Fehler:** "Import fehlgeschlagen" oder "Dateiformat nicht unterstützt"

**Lösungen:**
1. **Dateiformat prüfen:** Verwenden Sie nur CSV- oder Excel-Dateien
2. **Kampagnenauswahl überprüfen:** Stellen Sie sicher, dass eine Kampagne für den Import ausgewählt ist
3. **Dateigrößenlimits:** Halten Sie Dateien unter 10MB für optimale Leistung
4. **Kodierungsprobleme:** Speichern Sie CSV mit UTF-8-Kodierung

### Keine Leads importiert
**Problem:** Import wird abgeschlossen, aber keine Leads erscheinen

**Lösungen:**
1. **Spaltenüberschriften:** Überprüfen Sie, ob Überschriften dem erforderlichen Format entsprechen (phone, name, etc.)
2. **Datenvalidierung:** Überprüfen Sie, ob Telefonnummern im richtigen Format vorliegen
3. **Leere Zeilen:** Entfernen Sie leere Zeilen aus der Datei
4. **Erforderliche Felder:** Stellen Sie sicher, dass die Telefonnummernspalte gültige Daten enthält

### Teilerfolg beim Import
**Problem:** Einige Leads wurden importiert, andere sind fehlgeschlagen

**Lösungen:**
1. **Fehlerbericht herunterladen:** Überprüfen Sie fehlgeschlagene Zeilen auf spezifische Probleme
2. **Telefonnummernformat:** Stellen Sie internationales Format sicher (+1234567890)
3. **Duplikatserkennung:** Entfernen Sie doppelte Telefonnummern
4. **Ungültige Daten:** Beheben Sie fehlerhafte Telefonnummern oder fehlende erforderliche Daten

### Verarbeitungsverzögerungen
**Problem:** Import wird für längere Zeit als "in Bearbeitung" angezeigt

**Erklärung:**
- Große Dateien werden im Hintergrund verarbeitet
- Verarbeitungszeit steigt mit Dateigröße
- System reiht mehrere Importe ein

**Lösungen:**
1. **Auf Abschluss warten:** Überprüfen Sie Benachrichtigungen auf Updates
2. **Große Dateien aufteilen:** In kleinere Chargen aufteilen (500-1000 Leads)
3. **Support kontaktieren:** Wenn Verarbeitung 30 Minuten überschreitet

## Lead-Management-Probleme

### Lead erscheint nicht in Kampagne
**Problem:** Importierter Lead wird nicht in Kampagne angezeigt

**Prüfen:**
- Lead-Status (sollte "Created" oder "Scheduled" sein)
- Kampagnenfilter oder Sucheinstellungen
- Lead versehentlich als abgeschlossen markiert

**Lösungen:**
1. **Lead-Status zurücksetzen:** Status auf "Created" ändern, um erneut einzureihen
2. **Kampagnenzuweisung überprüfen:** Prüfen Sie, ob Lead der richtigen Kampagne zugewiesen ist
3. **Oberfläche aktualisieren:** Kampagnenseite neu laden

### Ungültige Telefonnummern
**Problem:** Telefonnummern werden beim Import abgelehnt

**Häufige Formate:**
- **Korrekt:** +1234567890 (internationales Format)
- **Falsch:** (123) 456-7890, 123-456-7890, 1234567890

**Lösungen:**
1. **Format standardisieren:** Alle Nummern ins Format +1234567890 konvertieren
2. **Formatierung entfernen:** Leerzeichen, Klammern, Bindestriche entfernen
3. **Ländercodes:** Entsprechende Ländercode-Präfixe hinzufügen
4. **Validierungstools:** Telefonvalidierung vor Import verwenden

### Doppelte Leads
**Problem:** Dieselbe Person erscheint mehrmals

**Prävention:**
1. **Datenbereinigung:** Duplikate vor Import entfernen
2. **Telefonvalidierung:** Auf Formatvariationen derselben Nummer prüfen
3. **Namensabgleich:** Potenzielle Duplikate nach Name + Telefon identifizieren
4. **Lead-Deduplizierung:** Tools zur Identifizierung ähnlicher Einträge verwenden

### Sekundäre Kontakte funktionieren nicht
**Problem:** Sekundäre Kontakte werden nicht angerufen

**Anforderungen:**
- Sekundärer Kontakt muss gültige Telefonnummer haben
- Primärer Kontakt muss zuerst max. Wiederholungen erreichen
- Kampagne muss für sekundäre Kontakte konfiguriert sein

**Lösungen:**
1. **Einrichtung überprüfen:** Konfiguration sekundärer Kontakte prüfen
2. **Lead-Status:** Sicherstellen, dass primärer Kontakt versucht wurde
3. **Kampagneneinstellungen:** Bestätigen, dass Anrufen sekundärer Kontakte aktiviert ist

## Variablen- und Datenprobleme

### Variablenzuordnungsfehler
**Problem:** Benutzerdefinierte Variablen werden nicht korrekt importiert

**Lösungen:**
1. **Spaltenüberschriften:** Müssen exakt mit [Assistentenkonfiguration](/ai-assistants/settings/prompt-and-tools#call-variables) übereinstimmen
2. **Datentypen:** Stellen Sie sicher, dass Daten den erwarteten Variablentypen entsprechen
3. **Erforderliche Variablen:** Alle im Assistenten definierten Variablen einschließen
4. **Standardwerte:** Überprüfen Sie, ob Standardwerte angewendet werden

### Fehlende Lead-Informationen
**Problem:** Leads fehlen Namen, E-Mails oder benutzerdefinierte Daten

**Lösungen:**
1. **Spaltenzuordnung:** Überprüfen Sie, ob alle Spalten korrekt zugeordnet sind
2. **Optionale Felder:** Bestätigen Sie, dass optionale Felder im Import enthalten sind
3. **Datenvalidierung:** Überprüfen Sie Quelldaten auf Vollständigkeit
4. **Erneuter Import:** Fehlende Daten hinzufügen und bei Bedarf erneut importieren

## Kampagnenintegrationsprobleme

### Lead-Statusprobleme
**Problem:** Leads stecken im falschen Status fest

**Statusbedeutungen:**
- **Created:** Bereit zum Anrufen
- **Scheduled:** Für nächsten Versuch eingeplant
- **Processing:** Wird gerade angerufen
- **Completed:** Fertig oder Ziel erreicht
- **Max Retries:** Wiederholungslimit erreicht

**Lösungen:**
1. **Manueller Statuswechsel:** Status in Leads-Tabelle aktualisieren
2. **Kampagnenneustart:** Kampagne bei Bedarf pausieren und neu starten
3. **Wiederholungseinstellungen:** Max. Wiederholungen oder Intervalle anpassen

### Probleme mit Variablenvererbung
**Problem:** Lead-Variablen funktionieren nicht in Anrufen

**Lösungen:**
1. **Assistenten-Einrichtung:** Variablen in [Assistenteneinstellungen](/ai-assistants/settings/prompt-and-tools#call-variables) definiert überprüfen
2. **Variablennamen:** Exakte Übereinstimmung zwischen Lead-Daten und Assistentenvariablen sicherstellen
3. **Datenformat:** Überprüfen Sie, ob Variablendaten im erwarteten Format vorliegen
4. **Testanrufe:** Manuelle Testanrufe verwenden, um Variablenübergabe zu überprüfen

## Dateiformatprobleme

### CSV-Probleme
**Häufige Probleme:**
- Falsches Trennzeichen (Komma verwenden, nicht Semikolon)
- Fehlende Überschriften in erster Zeile
- Zitierte Felder mit nicht maskierten Anführungszeichen
- Gemischte Kodierung (UTF-8 verwenden)

**Lösungen:**
1. **Standardformat:** Kommagetrennten Werte verwenden
2. **Überschriften erforderlich:** Erste Zeile muss Spaltennamen enthalten
3. **Textqualifikation:** Doppelte Anführungszeichen für Textfelder verwenden
4. **Kodierung:** Als UTF-8-kodierte CSV speichern

### Excel-Probleme
**Häufige Probleme:**
- Mehrere Blätter (nur erstes Blatt wird importiert)
- Verbundene Zellen verursachen Datenausrichtungsprobleme
- Formeln statt Werte
- Versteckte Zeichen oder Formatierung

**Lösungen:**
1. **Einzelnes Blatt:** Nur erstes Arbeitsblatt verwenden
2. **Zellen trennen:** Verbundene Zellen trennen
3. **Nur Werte:** Formeln in Werte konvertieren
4. **Sauberes Format:** Spezielle Formatierung entfernen

## Hilfe erhalten

### Vor Kontaktaufnahme mit Support
1. **Fehlerbericht herunterladen:** Spezifische Fehlermeldungen überprüfen
2. **Dateiformat prüfen:** CSV/Excel-Formatkonformität überprüfen
3. **Kleine Charge testen:** Zuerst 5-10 Leads importieren versuchen
4. **Dokumentation überprüfen:** Importanforderungen prüfen

### Bereitzustellende Informationen
- Erhaltene Fehlermeldungen
- Beispiel der Importdatei (mit entfernten sensiblen Daten)
- Kampagneneinstellungen und Anforderungen
- Erwartetes vs. tatsächliches Verhalten
- Assistentenvariablenkonfiguration
