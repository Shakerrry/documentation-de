---

## 1. Eigene Nummer mitbringen (SIP-Integration)

- **Ja**: Wenn Sie eine PBX oder einen VOIP-Anbieter haben, können Sie über einen SIP-Trunk integrieren.
- Siehe unseren [SIP-Integrationsleitfaden](/provisioning/sip-trunking/sip-integration) für schrittweise Anweisungen zur Einrichtung.
- Eingehende und ausgehende Anrufe laufen über Ihre bestehenden Leitungen, nutzen aber weiterhin die KI von Autocalls.ai.

## 2. Anrufer-ID-Verifizierung

- Wenn Sie nur möchten, dass Ihre bestehende Nummer für ausgehende Anrufe angezeigt wird, verifizieren Sie sie als **Anrufer-ID**.
- Dies verarbeitet jedoch keine eingehenden Anrufe, es sei denn, Sie verwenden SIP oder eine direkte Portierung.

## 3. Twilio-Konto

- Die direkte Integration von Twilio-Konten steht auf unserer Roadmap oder ist für einige Unternehmenskunden im Early Access verfügbar.
- In der Zwischenzeit können Sie das Beste aus beiden Welten haben, indem Sie Twilio-basierte Nummern über Autocalls.ai bereitstellen oder einen benutzerdefinierten Ansatz verwenden.

---

**Hinweis:** Wenn Sie unbedingt Ihr persönliches Twilio-Subkonto verbinden müssen, wenden Sie sich an den Support, um zu prüfen, ob Sie für das Pilotprogramm berechtigt sind.
