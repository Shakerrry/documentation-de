## Cal.com einrichten

1. Gehen Sie zur [Cal.com](https://cal.com/) Website und melden Sie sich für ein Konto an
2. Erstellen Sie einen neuen Event-Typ, zum Beispiel "Meeting mit KI-Assistent"
3. Kopieren Sie die Event-ID, indem Sie auf den Event-Typ klicken und diese Nummer aus der URL kopieren

<img height="200" src="/resources/cal-event-id.png" />

4. Erstellen Sie einen API-Schlüssel auf Cal.com

## Das Tool zu Ihrem Assistenten hinzufügen

Fügen Sie dieses Tool zu Ihrem Assistenten hinzu und übergeben Sie die Event-ID und den API-Schlüssel.

<img height="200" src="/resources/cal-tool.png" />
