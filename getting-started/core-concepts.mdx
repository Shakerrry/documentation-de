Willkommen bei den Grundlagen von Website Excellence Builders.ai. Hier erkunden wir die wesentlichen Bausteine, die Sie verstehen müssen, bevor Sie Ihren KI-Assistenten und Anrufe konfigurieren.

## 1. System-Prompt & Anfangsnachricht

**System-Prompt**

- Das "Gehirn" Ihres KI-Assistenten. Sie schreiben eine kurze (oder lange) Reihe von Anweisungen, die das Verhalten und den Tonfall des Assistenten leiten.
- Dies umfasst alle wichtigen Geschäftsinformationen, Haftungsausschlüsse oder wie Sie möchten, dass die KI mit Grenzfällen umgeht.
- Zum Beispiel: _"Sie sind ein hilfsbereiter Telefonassistent für die Firma XYZ, Sie begrüßen freundlich, bestätigen die Informationen des Anrufers usw."_

**Anfangsnachricht**

- Das allererste, was die KI sagt.
- Typischerweise kurz, wie "Hallo, danke für Ihren Anruf bei der Firma XYZ!"

## 2. Stimmen

- Website Excellence Builders.ai integriert sich mit **ElevenLabs**-Stimmen für natürlich klingende Sprache.
- Sie können Ihre eigene Stimme **klonen**, indem Sie Audioaufnahmen bereitstellen, oder aus integrierten Stimmen wählen.
- Jede Stimme kann anpassbare Parameter haben wie Sprechgeschwindigkeit, Füllwortverwendung oder Akzent (falls verfügbar).

## 3. Alles zusammenfügen

- Ein **KI-Assistent** wird erstellt, indem ein System-Prompt, eine Anfangsnachricht und eine gewählte Stimme kombiniert werden.
- Optional fügen Sie Tools hinzu (wie "Anruf weiterleiten" oder "Anruf beenden").
- Sobald der Assistent eingerichtet ist, weisen Sie ihn einer Telefonnummer zu (für eingehend) oder einer Kampagne (für ausgehend).
