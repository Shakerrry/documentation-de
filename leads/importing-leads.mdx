Importieren Sie große Mengen von Leads aus CSV- oder Excel-Dateien mit automatischer Validierung, Hintergrundverarbeitung und umfassender Fehlerberichterstattung.

## Import-Prozess-Übersicht

Das Lead-Import-System unterstützt:
- **CSV- und Excel-Dateien** (.csv, .xlsx, .xls)
- **Hintergrundverarbeitung** für große Dateien
- **Automatische Validierung** mit Fehlerberichterstattung
- **Sekundäre Kontakte** im selben Import
- **Variablenzuordnung** aus Assistentenkonfiguration

## Vorbereitung Ihrer Import-Datei

### Erforderliche Spalten
Ihre Import-Datei muss enthalten:
- **phone_number:** Primäre Kontakttelefonnummer (erforderlich)
- **Variablen:** Spalten, die mit Ihren [konfigurierten Variablen des Assistenten](/ai-assistants/settings/prompt-and-tools#call-variables) übereinstimmen

### Optionale Spalten
- **Sekundäre Kontaktspalten:** Zusätzliche Telefonnummern
- **Variablenüberschreibungen:** Benutzerdefinierte Werte für spezifische Leads

### Dateiformat-Beispiel
```csv
phone_number,customer_name,company,budget_range,secondary_phone_1,secondary_phone_2
+14155551234,John Smith,ABC Corp,10k-50k,+14155551235,+14155551236
+14155557890,Jane Doe,XYZ Inc,50k-100k,+14155557891,
+14155554567,Bob Johnson,Tech Start,5k-10k,,
```

## Kampagnenauswahl

### Zielkampagne wählen
1. **Kampagne auswählen** aus Dropdown während Import
2. **Assistentenvariablen** bestimmen automatisch erwartete Spalten  
3. **Variablenvererbung** wendet Assistenten-Standards auf alle importierten Leads an

### Variablenzuordnung
- **Assistentenvariablen** werden zu Spaltenüberschriften in Beispieldatei
- **Standardwerte** aus Assistentenkonfiguration
- **Überschreibungsfähigkeit** mit benutzerdefinierten Werten pro Lead

<Note>
**Beispieldatei herunterladen:** Der Import-Dialog bietet eine Beispiel-CSV mit korrekten Kopfzeilen basierend auf den Assistentenvariablen Ihrer ausgewählten Kampagne.
</Note>

## Import-Schritte

### 1. Import-Prozess starten
1. **Navigieren Sie zum Leads-Bereich** oder zum Kampagnen-Leads-Tab
2. **Klicken Sie auf "Leads importieren"**
3. **Wählen Sie Kampagne** für Lead-Zuweisung
4. **Beispieldatei herunterladen**, um das erwartete Format zu sehen

### 2. Datei hochladen
1. **Datei auswählen:** Wählen Sie Ihre CSV- oder Excel-Datei
2. **Fehler ignorieren-Option:** Aktivieren, um ungültige Zeilen zu überspringen
3. **Einstellungen überprüfen:** Kampagne und Optionen bestätigen
4. **Import starten:** Klicken Sie auf "Jetzt importieren"

### 3. Hintergrundverarbeitung
- **Warteschlangenverarbeitung:** Große Dateien werden im Hintergrund verarbeitet
- **Fortschrittsverfolgung:** Import-Status wird automatisch aktualisiert
- **Benachrichtigung:** E-Mail-Benachrichtigung bei Import-Abschluss
- **Fehlerberichterstattung:** Herunterladen eines Berichts über fehlgeschlagene Zeilen

## Validierungsregeln

### Telefonnummern-Validierung
- **Formatprüfung:** Internationales Format bevorzugt (+1234567890)
- **Duplikaterkennung:** Verhindert doppelte Telefonnummern
- **Ungültige Nummern:** Automatisch markiert und übersprungen

### Variablen-Validierung
- **Erforderliche Felder:** Als erforderlich markierte Variablen müssen Werte haben
- **Datentypen:** Text-, Zahlen-, Boolean-Validierung wo zutreffend
- **Längenbeschränkungen:** Maximale Zeichenlimits werden durchgesetzt

### Validierung sekundärer Kontakte
- **Telefonformat:** Gleiche Validierung wie primäre Nummern
- **Einzigartige Nummern:** Sekundäre Kontakte müssen sich vom Primärkontakt unterscheiden
- **Variablenkonsistenz:** Variablen sekundärer Kontakte werden unabhängig validiert

## Fehlerbehandlung

### Ungültige Zeilen überspringen
Wenn "Fehler ignorieren" aktiviert ist:
- **Gültige Zeilen importiert:** Erfolgreich validierte Leads werden hinzugefügt
- **Ungültige Zeilen übersprungen:** Problemzeilen werden vom Import ausgeschlossen
- **Fehlerbericht generiert:** Detaillierten Bericht der übersprungenen Zeilen herunterladen
- **Teilerfolg:** Import wird mit gültiger Teilmenge abgeschlossen

### Häufige Validierungsfehler
- **Ungültige Telefonnummern:** Falsches Format oder unmögliche Nummern
- **Fehlende erforderliche Felder:** Leere Werte für erforderliche Variablen
- **Doppelte Telefonnummern:** Telefonnummer existiert bereits im System
- **Dateiformat-Probleme:** Beschädigte oder nicht unterstützte Dateiformate

### Fehlerbericht herunterladen
Nach Import-Abschluss:
- **Download-Link** in Benachrichtigung bereitgestellt
- **Detaillierte Fehlerbeschreibungen** für jede fehlgeschlagene Zeile
- **Korrekturhilfe** zum Beheben von Problemen
- **Erneuter Import möglich** mit korrigierten Daten

## Import sekundärer Kontakte

### Mehrere Telefonnummern pro Lead
Fügen Sie zusätzliche Spalten für sekundäre Kontakte hinzu:
```csv
phone_number,customer_name,secondary_phone_1,secondary_phone_2,secondary_phone_3
+14155551234,John Smith,+14155551235,+14155551236,+14155551237
```

### Variablen für sekundäre Kontakte
Fügen Sie spezifische Variablen für sekundäre Kontakte hinzu:
```csv
phone_number,customer_name,secondary_phone_1,secondary_1_name,secondary_1_role
+14155551234,John Smith,+14155551235,Jane Smith,Entscheidungsträger
```

### Kontakt-Reihenfolge
- **Primärkontakt:** Wird immer zuerst angerufen
- **Sekundäre Reihenfolge:** Werden in Spaltenreihenfolge angerufen (secondary_phone_1, dann secondary_phone_2, etc.)
- **Kampagnen-Koordination:** Respektiert Kampagnen-Wiederholungseinstellungen zwischen Kontakten

## Handhabung großer Dateien

### Hintergrundverarbeitung
- **Automatische Warteschlange:** Dateien mit über 1000 Zeilen werden im Hintergrund verarbeitet
- **Speichereffizienz:** Streaming-Verarbeitung verhindert Timeouts
- **Fortschrittsverfolgung:** Echtzeit-Statusaktualisierungen
- **Systemleistung:** Keine Auswirkung auf Plattformleistung während Import

### Benachrichtigungssystem
- **E-Mail-Benachrichtigungen:** Gesendet bei Import-Abschluss
- **Erfolgszusammenfassung:** Anzahl erfolgreich importierter Leads
- **Fehlerzusammenfassung:** Anzahl und Details fehlgeschlagener Zeilen
- **Download-Links:** Zugriff auf Fehlerberichte und importierte Daten

### Verarbeitungszeit
- **Kleine Dateien (< 100 Zeilen):** Normalerweise innerhalb 1 Minute abgeschlossen
- **Mittlere Dateien (100-1000 Zeilen):** 2-5 Minuten typisch
- **Große Dateien (1000+ Zeilen):** 10-30 Minuten je nach Komplexität
- **Sehr große Dateien (10k+ Zeilen):** Kann mehrere Stunden dauern

## Best Practices

- **Beispieldatei verwenden:** Bereitgestellte Vorlage für korrektes Format herunterladen
- **Telefonformat:** Internationales Format ohne Trennzeichen (+1234567890)
- **Saubere Daten:** Leere Zeilen entfernen und Inhalt validieren
- **Zuerst testen:** Mit kleiner Batch beginnen, um Format zu verifizieren
- **Variablen abgleichen:** Sicherstellen, dass Spalten mit Assistentenkonfiguration übereinstimmen
